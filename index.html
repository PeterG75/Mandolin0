Introduction.

Mandolin0 is a web form bruteforcing tool created with the aims to be very performant and easy to use. It is based on the concept of Template and Oracle, this will help not experienced people to use the tool without the needed knowledge of how to setup the login HTTP Post (this of course isn't an excute to not learn hot the things work).

You can download a compiled version here.

What are Template and Oracle

Template is an abstract HTTP request that are used by a given website to login a user. The aim of the project is to provide templates for the most common used open source platform, in order to avoid to known how to bruteforce a given application. However if you have to bruteforce a custom application you have to create a custom template. All template are stored in the Templates directory. You need to edit the template file and insert the placeholder {USERNAME} and {PASSWORD} at the correct place. See the Default template (located at Templates/Default/Default.txt) to better understand how a template works.

Oracle is used to known if a given test has successful discovered the password or not. It is an XML file where you can configure a list of text that must be matched or not. See the Default oracle (located at Oracles/Default/Default.txt) to better understand how a oracle works.
